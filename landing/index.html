Quickstart

Terminal 
# Install a workflow
antfarm workflow install ./workflows/feature-dev

# Run a task
antfarm workflow run feature-dev "Ship the onboarding prompt"

# Check status
antfarm workflow status "Ship the onboarding prompt"

How It Works

1

Define your workflow

Write a workflow.yml with agents, steps, and context. Each agent gets a role, skills, and workspace files.

2

Install it

Antfarm provisions agent workspaces, installs skills, and registers the workflow. One command.

3

Run a task

Give it a task title. Antfarm creates a run, passes context between steps, and each agent picks up work on its own schedule.

4

Ship it

Agents execute their steps ‚Äî planning, implementing, reviewing ‚Äî passing context forward until the pipeline completes.

Commands

antfarm workflow install <source>

Install a workflow from a local path, GitHub repo, or raw URL.

antfarm workflow run <id> <task>

Start a new run with the given task title.

antfarm workflow status <task>

Check the status of a running or completed task.

antfarm workflow update <id>

Update a workflow to the latest version from its source.

antfarm workflow uninstall <id>

Remove a workflow and all its associated resources.
